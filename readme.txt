The aim of this project is create and deploy an Adevnture game website, which focuses on 
a selected scenario, multiple opations are provided to divert the scenario to a different
ending. Pictures and different color schemes are used for User Expeience.


Functionality:

Dark Mode - enabled by the user and will be stored in the local storage of the 
	    browser the next time the user visits it will automatically apply it. if dark
	    mode was enabled when it was closed.


Button Navigation - foward navigation is done by the use of buttons to divert the 
		    scenario to different endings, while going back to the previous
		    the user needs to rely on the browser for providing a back button.
